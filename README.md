## 🚀 Getting Started

Clone the repository

```bash
git clone https://github.com/abhi-yanta/-DSA-Zamming-.git
```

Move into the project

```bash
cd -DSA-Zamming-
```

Compile any file

```bash
g++ filename.cpp -o output
```

Run

### Linux / macOS

```bash
./output
```

### Windows

```bash
output.exe
```
