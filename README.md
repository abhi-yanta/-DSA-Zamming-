# 🚀 DSA-Zamming

A collection of **Data Structures & Algorithms (DSA)** problems solved in **C++**.

This repository is part of my DSA learning journey, where I regularly practice coding problems, improve problem-solving skills, and build a strong foundation for coding interviews and competitive programming.

## ▶️ How to Run

Compile using g++:

```bash

./output
```

Example:

```bash
g++ 05gcd.cpp -o gcd
./gcd
```
